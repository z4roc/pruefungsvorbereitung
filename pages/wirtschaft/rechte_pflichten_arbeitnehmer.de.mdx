# Rechte und Pflichten von Vertragsparteien

| Rechte des Arbeitnehmers     | Rechte des Arbeitgebers |
| ---------------------------- | ----------------------- |
| - Sicherheit am Arbeitsplatz | - Schweigepflicht       |
| - Vergütungspflicht          | - Arbeitspflicht        |
| - Zeugnispflicht             | - Sorgfaltspflicht      |
| - Beschäftigungspflicht      | - Wettbewerbsverbot     |
|                              | - Gehorsamspflicht      |

## Rechte und Pflichten bei Azubis

| Pflichten des Ausbilders (Arbeitgeber) | Pflichten des Azubis (Arbeitnehmer) |
| -------------------------------------- | ----------------------------------- |
| - Sicherheit am Arbeitsplatz           | - Schweigepflicht                   |
| - Vergütungspflicht                    | - Arbeitspflicht                    |
